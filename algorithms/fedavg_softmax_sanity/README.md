# FedAvg Softmax Sanity Baseline

This folder documents the first lightweight FedAvg sanity baseline.

Current implementation:

`kuhar_delivery/scripts/train_kuhar_fedavg_numpy.py`

Result note:

`kuhar_delivery/reports/kuhar/kuhar_fedavg_softmax_baseline_v1.md`

This baseline uses statistical accelerometer features and a softmax classifier.
It is useful for checking the pipeline, but it is not the neural HAR baseline.
